# AI Safety Governance Platform

> **NYU Capstone Project** in collaboration with the **UNICC AI Safety Lab**

## Project Overview

This project presents an AI Safety Governance Platform designed to evaluate AI systems through a multi-layer governance workflow. The platform integrates multiple expert modules, an arbitration layer, and governance rules to improve transparency, explainability, and decision-making in AI safety evaluation.


## My Contributions

As a member of **Project 3 – Governance Layer**, I was responsible for:

- Designing the Governance Layer workflow
- Designing the overall system interaction flow
- Developing the Streamlit interactive dashboard
- Building the visualization for:
  - Expert module evaluations
  - Arbitration results
  - Routing Trace
  - Audit Log
  - JSON output
- Participating in system integration and testing
- Supporting governance workflow design and explainability
